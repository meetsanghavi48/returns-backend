# Returns Manager Backend

Shopify OAuth backend for Returns Manager app.

## Deploy to Render

Set these Environment Variables in Render:
- `SHOPIFY_API_KEY` = c1542c4ed17151e558edc3f37ceb9fd2
- `SHOPIFY_API_SECRET` = your_client_secret
- `SHOP_DOMAIN` = your-store.myshopify.com
- `BACKEND_URL` = https://your-render-app.onrender.com

## After Deploy

1. Visit: https://your-render-app.onrender.com/auth?shop=your-store.myshopify.com
2. Approve the install in Shopify
3. You're connected!
